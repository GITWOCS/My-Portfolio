Django Development

A couple of months ago, I enrolled in a Python Programming full course on Udemy. During a full and immersed period of training, I developed my proficiency in Python and I added the mastery of a new language to my portfolio. During the course of the training, I was able to develop a good number of projects related to web development, application development and more over. One of my project is the later. A website for a restorant.
For this project, I used the Python Programming Language and Django technology to develop this platform. Including a welcome page, this platform provides an appealing and very attracting interface to its users. In the landing page, you are provided a menu you can choose from depending on the price, the name, the ingredients and so on. Added to that, due to the Django technology, this platform provides the administrator with an amazing interface to manage the restaurant business. With this interface, you can easily add a new pizza, modify, delete etc...
With the use of these amazing tool which are Python and Django, I intend to do much more work and improve in my knowledge.
